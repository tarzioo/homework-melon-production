Don't Get Lost!

It is important that you don't get bogged down in the sort of "black-box"-yness of the methods here. This is largely about learning to function within a somewhat complicated object oriented script. EXTRA CREDIT: We've been learning Object Oriented Programming (OOP) here at Hackbright, but you may (or may not) have heard of another programming paradigm called Functional Programming (FP). If you're interested in getting a better sense of what that means read this: http://www.codenewbie.org/blogs/object-oriented-programming-vs-functional-programming
